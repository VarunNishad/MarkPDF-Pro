# MarkPDF Pro

**MarkPDF Pro** is a modern, professional Markdown to PDF converter built with React, Vite, and Shadcn UI. It offers a real-time preview and print-optimized PDF export.

## ‚ú® Features

- **Real-time Preview**: Type markdown on the left, see the formatted document on the right instantly.
- **Print-Optimized**: Exports beautiful, A4-formatted PDFs using your browser's native print engine.
- **Privacy Focused**: Zero-server data storage. Your Markdown content stays purely local in your browser.
- **Modern UI**: Built with [shadcn/ui](https://ui.shadcn.com/) and [Tailwind CSS v4](https://tailwindcss.com/) for a polished look.

## üöÄ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

    ```bash
    git clone https://github.com/yourusername/markpdf-pro.git
    cd markpdf-pro
    ```

2. **Install dependencies**

    ```bash
    npm install
    ```

3. **Start the development server**

    ```bash
    npm run dev
    ```

4. **Open the app**
    Visit `http://localhost:3000` (or the port shown in your terminal).

## üõ† Usage

1. **Write Markdown**: Use the editor on the left to write or paste your content. You can also import `.md` files directly.
2. **Preview**: The right panel shows exactly how your document will look.
3. **Export**: Click the **Print** button in the top right. In the print dialog, choose **"Save as PDF"** as the destination.
